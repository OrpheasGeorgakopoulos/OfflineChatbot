
```bash
python main.py
```

---

## 🧠 How It Works

Instead of using simple keyword matching, the chatbot uses:

* Sentence Transformers
* Semantic similarity search
* Vector embeddings
* Cosine similarity scoring

This allows users to ask questions in different ways and still receive relevant answers.
